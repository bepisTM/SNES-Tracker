ID666 は、曲の情報を SPC ファイルに格納するための拡張規格です。 ID666 is an extension standard for storing information of the song in the SPC file.

SNES SPC700 Player は、 Nitro 氏が立案された ID666 規格に対応しています。 SNES SPC700 Player is compatible with ID666 standards Nitro Mr. has been planning. ID666 を利用すれば、曲名などの基本情報の他、演奏時間、フェードアウト時間を曲別に指定することができます。 By using the ID666, other basic information, such as song title, play time, you can specify the fade-out time for each song.

ID666 のフォーマットは、テキスト フォーマットとバイナリ フォーマットの 2 種類が存在します。 ID666 format, there are two types of text format and binary format. SNES SPC700 Player は両方のフォーマットをサポートしています。 SNES SPC700 Player supports both formats.

この資料では ID666 のフォーマットについての詳細を説明しています。 This article we have details about the format of the ID666.
